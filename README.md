# 🩸 Blood Connect

A full-stack web application that connects blood donors with recipients, enabling quick and reliable access to blood during emergencies. The platform simplifies donor registration, blood requests, and communication while promoting voluntary blood donation.

---

## 🚀 Features

- 👤 User Registration & Login
- 🩸 Blood Donor Registration
- 🔍 Search Donors by Blood Group
- 🚨 Emergency Blood Requests
- 🏥 Hospital Integration
- 📍 Location-Based Donor Search
- 📱 SMS Notification Support
- 📋 Donor Management Dashboard
- 💻 Responsive User Interface

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/BloodConnect.git
```

### 2. Navigate to the project

```bash
cd BloodConnect
```

### 3. Install dependencies

```bash
npm install
```

### 4. Configure Environment Variables

Create a `.env` file and add:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
```

### 5. Start the server

```bash
npm start
```

The application will run on:

```
http://localhost:5000
```

## 📖 Workflow

1. User registers or logs in.
2. Donors register their blood details.
3. Recipients search donors by blood group.
4. Emergency requests are created.
5. Nearby eligible donors receive notifications.
6. Donors can respond and connect with recipients.
